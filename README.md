## 📦 Jak zainstalować na Windows

### Opcja 1: Plik `.exe`
1. Pobierz plik z sekcji Releases.
2. Otwórz folder Pobrane.
3. Kliknij plik dwa razy.
4. Jeśli Windows poprosi o zgodę, potwierdź.
5. Postępuj zgodnie z oknem instalacji lub startu.

### Opcja 2: Plik `.msi`
1. Pobierz plik instalacyjny.
2. Kliknij go dwa razy.
3. Przejdź przez kroki instalacji.
4. Po zakończeniu uruchom aplikację z menu Start lub skrótu na pulpicie.

### Opcja 3: Plik `.zip`
1. Pobierz archiwum.
2. Kliknij je prawym przyciskiem myszy.
3. Wybierz opcję rozpakowania.
4. Otwórz folder po rozpakowaniu.
5. Uruchom plik aplikacji.
